YOLO

作業簡介

先標記100張圖片作為訓練資料，用來fine tune YOLO，最後得到標記好的圖片。

使用技術 / 環境

  • Language: Python 3.9
  • pre train model: YOLO11s
  • Dataset: 作業提供的100張圖片

流程

1.標記資料

下載labelImg並標記完所有圖片

2. 決定資料增強參數

條參的重點在想辦法增強模型對小物件(球， 球拍)的準確度，推測是由於樣本數相較其他更少且占據得像素也很小，高速飛行的球相當模糊，肉眼也不好分辨，因此嘗試提高解析度並且多拼貼不同照片，看看能不能有更多球及球拍能訓練。

hyper parameter

hyp_config = {
    "epochs": 500,          
    "imgsz": 1280, #從640調到1280，但影響不大         
    "batch": 8,             
    "device": 0,            
    "workers": 2,        
    "optimizer": "auto",
    "patience": 50,   

    #做一些輕微旋轉
    "degrees": 10.0,      
    "translate": 0.1,       
    "scale": 0.5,           
    "shear": 5.0,            

    #拼貼影像，看看能不能有更多小物件的資料
    "mosaic": 0.8,         
    "mixup": 0.1,          
    "copy_paste": 0.2,     

    #資料不可能會出現上下顛到的球桌，所以關閉垂直旋轉
    "fliplr": 0.5,         
    "flipud": 0.0,    
    
}

實作成果

可以看到模型對大物件(球桌，裁判和球員)的正確率相當高，但在小物件(球拍，球)的判定上還是不太準確，可能需要更多訓練資料才能進步。
